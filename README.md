# Omarchy Sir Williams Theme

Sir Williams is an Omarchy theme inspired by classic gentleman clubs, deep ocean blues, antique brass, dark wood, and nautical libraries.

## Status

Initial usable version. The theme currently includes the core Omarchy palette, Hyprland window styling, Waybar glass styling, and a NASA/Webb M82 infrared background.

## Installation

```bash
omarchy theme install https://github.com/afwilliams/omarchy-sir-williams-theme.git
```

## Usage

```bash
omarchy theme set "Sir Williams"
```

## Theme Direction

- Deep ocean navy background
- Warm ivory foreground
- Antique brass accent
- Muted bottle green, coral, plum, and sea glass terminal colors
- Classic club and nautical-library mood

## Backgrounds

Included wallpaper:

- `backgrounds/01-m82-starburst-infrared.jpg`

The background is an official James Webb Space Telescope image of M82, also known as the Cigar Galaxy. Credit: ESA/Webb, NASA & CSA, A. Bolatto.
